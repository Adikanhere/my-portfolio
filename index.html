Projects

LLM for Retail Business(Optimizing Clothing with AI)

LLM for Retail Business is an AI-driven, interactive assistant designed to help users ask natural language questions about their retail inventory data and get precise answers. It uses dynamically generated SQL queries to fetch relevant information. By leveraging Large Language Models (LLMs) with few-shot learning, the tool translates user input into SQL and retrieves data from a MySQL database making valuable business insights easily accessible to users without technical expertise.

E-commerce Web Application using Java Full Stack

Developed a full-stack e-commerce platform using React (frontend) and Spring Boot (backend) with features like product listing, shopping cart, user authentication, payment integration, and order tracking. Implemented secure REST APIs and MySQL database to manage transactions and user operations efficiently.

Full-Stack CRM Dashboard

A full-stack Customer Relationship Management (CRM) system with a React frontend and a Spring Boot / REST API backend, fully deployed online. It enables efficient management of Customers, Leads, Opportunities, and Users, featuring interactive tables, search functionality, and smooth animations using Framer Motion. The responsive dashboard visualizes historical data with Bar, Pie, and Doughnut charts, and user authentication ensures secure access. The interface is fully mobile-friendly and intuitive.

AgriClient

I built an Agriculture Marketplace Platform where farmers can directly connect with clients to sell crops, reducing their dependency on brokers. The platform includes features such as user login and registration, crop listing, weather updates, and a simple farmer-client chat system. The backend is developed using Spring Boot with JWT-secured APIs and MySQL, while the frontend is built with React and deployed successfully.

Hotel Management & Rating Service

Hotel Management & Rating Services built using Java, Spring Boot, Spring Security, and REST APIs, with integration of PostgreSQL, MySQL, and MongoDB for exploring multi-database handling. Applied Apache Kafka for learning real-time communication between services and practiced containerization using Docker & Kubernetes. Explored CI/CD pipeline concepts to understand automation and modern deployment workflows.

Product Inventory Management System

Developed a Product Inventory Management System to manage adding, viewing, updating, and deleting products. Applied core Spring concepts including dependency injection, bean lifecycle management, autowiring, annotations, expression language, and collection handling.
